Der Entwickler stellt hiermit ausdrücklich klar, dass er keinerlei Haftung für entstehende - auch zukünftige - Schäden an Computern, deren kabelgebundene oder kabellos verbundene Hardware übernimmt. Ebenso wird jeglicher Gewährleistungs- und Schadensersatzanspruch vollumfänglich abgelehnt.
Die Installation und insbesondere der Betrieb der Software geschieht auf eigene Gefahr. Der Entwickler gibt keinen Support für einzelne Störungen oder Fehlverhalten von Computern und ist dazu auch nicht verpflichtet.

Die Software darf von Privatpersonen auf unbegrenzt vielen Computern ohne zeitliche oder funktionelle Einschränkung genutzt werden. Die Software nimmt Audio auf unter Berücksichtigung der Wahrung der Rechte Dritter. Eine private Weitergabe von Soundbänken ist gestattet. Beachten Sie dabei, dass die Wahrung der Rechte Dritter gewahrt bleiben muss.
Die kommerzielle Nutzung der Software ist nur mit einer lizenzierten Version gestattet. Lizenzen erhalten Sie direkt beim Entwickler unter info@patchdirector.com.
